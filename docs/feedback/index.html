Security

Use the security path for vulnerabilities

Security reports should go through the private advisory flow instead of a public issue or ordinary email.

Report a vulnerability

What helps most.

  • Tell me which surface you used: Canary app, remote app beta, CLI, MCP server, install, or docs.
  • Include the ibkr version, OS, browser or MCP client, and whether the source was TWS or IB Gateway.
  • For bugs, include the shortest steps that reproduce it and the redacted output you saw.
  • For feature requests, describe the workflow and what good would look like.
